# Redmine-Plugin-wiki-related-issue
Project for the plugin which shows the related issue list of the wiki page of Redmine

When you link the wiki page in the issue page, the related issues are shown in the wiki page. It means you can see all the issues which refers the wiki page at once. 
It doesn't require any settings other than installation of the plugin. 

* Tested in Redmine 5.1.1.stable (Ruby 3.2.2-p53, Rails 6.1.7.6, and postgressql)
* Turn on the default settings in the plugin list of Redmine if it shows the related issue in the wiki page as default or not. (Still you can check or uncheck of each wiki page though)
* Show issue number, title, author, status, or so on in the related issue list
